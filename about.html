The Project

The project XOS or Experimental Operating System is a platform to help students build a simple Operating System. Its an instructional tool for students to learn and implement OS data structures and functionalities on a simulated machine called XSM (Experimental String Machine).

The OS is programmed using a customized language known as SPL (System Programmer's Language) and application programs which run on the OS is programmed using APL (Application Programmer's Language).

Contribute

XOS is maintained as an open-source project in Github http://github.com/xosnitc. There are several components of the XOS project which needs contributions from the community, which include

SPL (System Programmer's Language) - Design and Compiler Project Page
APL (Application Programmer's Language) - Design and Compiler Project Page
XFS (File System) Interface with UNIX Project Page
XSM (Experimental String Machine) Simulator Project Page
