SPL or System Programmer's Language is a high level language for programming XOS.

Prerequisites
  • GCC
  • Flex / Lex
  • Bison / Yacc
Using the compiler
1. make
2. ./spl < path-to-file


Download
You can download with Git using
git clone https://github.com/xosnitc/spl

or by clicking Download

APL or Application Programmer's Language is a high level language for writing application programs which run on XOS (eXperimental Operating System).

Prerequisites
  • GCC
  • Flex / Lex
  • Bison / Yacc
Using the compiler
1. make
2. ./apl < path-to-file


Download
You can download with Git using
git clone https://github.com/xosnitc/apl

or by clicking Download

XFS-Interface is used as an interface to access XFS Filesystem from UNIX. The interface has functionalities to load, remove, copy, and list files in an XFS formatted disk.

Prerequisites
  • GCC
Using the interface
1. make
2. ./fileSystem
3. Type "help" for list of commands.

Download
You can download with Git using
git clone https://github.com/xosnitc/xfs-interface

or by clicking Download

XSM Machine simulator simulates XSM hardware.

Prerequisites
  • GCC
  • Flex / Lex
  • Bison / Yacc
Running the machine simulator
1. make
2. ./xsm < path-to-disk-file


Download
You can download with Git using
git clone https://github.com/xosnitc/xsm

or by clicking Download
